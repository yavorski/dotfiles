# Setup Vim

## Process

*

```shell
mkdir ~/.vim
mkdir ~/.vim/autoload
mkdir ~/.vim/bitmaps
mkdir ~/.vim/colors
mkdir ~/.vim/swaps
mkdir ~/.vim/syntax
mkdir ~/.vim/undo
curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

## Fonts

* Not Automated

```shell
git clone https://github.com/ryanoasis/nerd-fonts
cd nerd-fonts
./install
cd ..
rm -rf nerd-fonts
```
